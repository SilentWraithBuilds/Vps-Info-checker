# SILENTWRAITHH VPS INFO PANEL

Advanced Linux VPS monitoring panel written in Bash.

Displays detailed system information with colors, ASCII art, network stats, hardware info, and live monitoring features.

---

## Features

* Beautiful ASCII Banner
* CPU Information
* RAM Usage
* Disk Usage
* Public & Local IP
* ISP & Location
* Internet Connectivity Check
* Top Processes
* Virtualization Detection
* Package Count
* Colored Terminal UI
* One-line Installer Support

---

## Installation

### Quick Install

```bash
bash <(curl -s https://YOUR-DOMAIN/script.sh)
```

or

```bash
curl -fsSL https://YOUR-DOMAIN/script.sh | bash
```

---

## Manual Installation

Clone repository:

```bash
git clone https://github.com/YOURUSERNAME/silentwraithh-vps.git
```

Go inside:

```bash
cd silentwraithh-vps
```

Give permissions:

```bash
chmod +x silentwraithh.sh
```

Run:

```bash
./silentwraithh.sh
```

---

## Requirements

* Ubuntu / Debian / Linux VPS
* bash

# 🖥️ SILENTWRAITHH VPS INFO CHECKER

Advanced Linux VPS monitoring panel written in Bash.

Displays detailed VPS/server information with a beautiful terminal UI, ASCII branding, hardware statistics, networking details, and live system monitoring.

---

# ✨ Features

* 🎨 Colored Terminal Interface
* 🔥 SILENTWRAITHH ASCII Banner
* 🧠 CPU Information
* ⚡ CPU Core Count
* 🟩 RAM Usage
* 💾 Disk Usage
* 🌍 Public IP Detection
* 🏠 Local IP Detection
* 📍 Country / Region / ISP
* 🧱 Virtualization Detection
* 📦 Installed Packages Count
* 📈 System Load
* 🌐 Internet Connectivity Check
* 👥 Logged In Users
* 🚀 Top Running Processes
* 🕒 Current Time & Timezone

---

# 🚀 Quick Install

Run directly without downloading:

```bash id="2yr8ic"
bash <(curl -s https://raw.githubusercontent.com/SilentWraithBuilds/Vps-Info-checker/main/main.sh)
```

OR

```bash id="20n1va"
curl -fsSL https://raw.githubusercontent.com/SilentWraithBuilds/Vps-Info-checker/main/main.sh | bash
```

---

# 📥 Manual Installation

Clone the repository:

```bash id="cwqx7m"
git clone https://github.com/SilentWraithBuilds/Vps-Info-checker.git
```

Go into the folder:

```bash id="e9m4a4"
cd Vps-Info-checker
```

Give execute permissions:

```bash id="4x8e6m"
chmod +x main.sh
```

Run the script:

```bash id="2h69zy"
./main.sh
```
